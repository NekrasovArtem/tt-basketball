# tt-basketball

Тестовое задание

## Installation

```
git clone https://github.com/NekrasovArtem/tt-basketball
cd tt-basketball
```

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
